In python, the stack operations like `push` and `pop` is in built

## Initialization

Go to your terminal and type `python` or `python3` as per your needs.

#### Stack Implementation

1. Making the list

```
mylist = []
```

2. Using the `push` method

```
mylist.append("value")
```

3. Using the `pop` method

```
mylist.pop()
```
